Main Features

Discover the powerful capabilities of HeatMon

1. Real-time CPU Temperature Monitoring

  • Displays the temperature for each CPU core (up to 16 cores).
  • Current, minimum, and maximum temperature information.
  • Progress bar visualization with color coding (green, orange, red).

2. Detailed Information System

  • Product Information: vendor, product name, ID, system UUID.
  • Motherboard Information: manufacturer, product, serial number, version.
  • BIOS Information: vendor, version, SMBIOS version, release date.
  • Operating System Information: name, version, build, architecture, installation date, last boot.
  • Processor Information: name, number of cores, vendor, socket.

3. Real-time Graph

  • Graphical display for monitoring temperature trends.
  • Ability to select specific cores to monitor.
  • Several display theme options (Light, Dark, Gradient).
  • Option to save graphs as images (PNG, JPEG, BMP).
  • Ability to print graphs directly.

4. Additional Features

  • Click to Copy: click on the information label to copy text to clipboard.
