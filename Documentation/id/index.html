Pendahuluan

HeatMon adalah alat pemantauan ringan yang dirancang untuk melacak suhu dan tegangan CPU secara real-time. Aplikasi ini menyediakan visualisasi grafik langsung, opsi laporan, dan tema yang dapat disesuaikan untuk meningkatkan kegunaan serta tampilan profesional.

Persyaratan Sistem

  • Sistem Operasi: Windows Vista / 7 / 8 / 10 / 11
  • CPU: x86/x64 dengan dukungan sensor suhu

Memulai

  1. Jalankan HeatMon.exe
  2. Jendela utama menampilkan:
    • Nama CPU dan jumlah inti
    • Suhu per inti (°C atau °F)
    • Tegangan CPU (Volt atau mV)
    • Grafik suhu real-time

Fitur Utama

  • Pemantauan suhu CPU real-time per inti
  • Pemantauan tegangan CPU (Volt / mV)
  • Grafik suhu interaktif dengan 12 tema warna
  • Ekspor laporan (TXT) dan grafik (PNG)
  • Mode "Tetap di Atas" untuk visibilitas berkelanjutan
  • Lisensi penggunaan Pribadi atau Komersial
  • Shortcut keyboard untuk akses lebih cepat

Menu dan Kontrol

Berkas:

  • Simpan Laporan TXT -> Simpan laporan suhu/tegangan
  • Simpan Grafik PNG -> Ekspor grafik sebagai PNG
  • Keluar -> Tutup HeatMon

Tampilan:

  • Tampilkan Inti Tambahan -> Tampilkan slot inti tambahan
  • Tetap di Atas -> Pertahankan HeatMon di atas jendela
  • Tema Grafik -> Ganti tema warna grafik
  • Tampilkan Grafik -> Tampilkan grafik

Bantuan:

  • Konten -> Buka file bantuan ini
  • Tentang -> Info aplikasi & kredit
  • Masukkan Serial -> Buka formulir registrasi

Opsi:

  • Tegangan: Volt atau mV
  • Suhu: °C atau °F

Registrasi:

  • Masukkan kunci -> Tempel/masukkan serial dari clipboard
  • Daftarkan -> Aktifkan lisensi komersial
  • Kontak -> Email dukungan untuk masalah lisensi

Shortcut Keyboard

  • F1 -> Buka Dokumentasi Bantuan
  • Ctrl + S -> Simpan Laporan (TXT)
  • Ctrl + Alt + S -> Simpan Grafik (PNG)
  • Ctrl + G -> Tampilkan / Sembunyikan Grafik

Interpretasi Data

  • Nama CPU -> Model prosesor
  • Jumlah Inti -> Jumlah inti yang terdeteksi
  • Tegangan -> Tegangan CPU (Volt atau mV)
  • Suhu -> Bacaan suhu per inti

Tips Penggunaan

  1. Aktifkan "Tetap di Atas" saat bermain game atau benchmarking
  2. Simpan laporan secara berkala untuk perbandingan performa
  3. Berhati-hatilah jika suhu CPU melebihi 90 °C
  4. Gunakan shortcut untuk alur kerja lebih cepat
  5. Registrasi untuk membuka mode Penggunaan Komersial

Pemecahan Masalah

  1. Tidak ada data suhu?
    • Pastikan CPU mendukung sensor suhu.
  2. RPM kipas tidak terdeteksi?
    • Tidak semua sistem menampilkan sensor kecepatan kipas.
  3. Grafik kosong?
    • Tunggu beberapa detik hingga data muncul.
  4. Aplikasi tertutup tiba-tiba?
    • Jalankan HeatMon sebagai Administrator.
  5. Registrasi gagal?
    • Periksa serial key lalu coba lagi.

Lisensi dan Registrasi

  • Mode Default: Hanya untuk Penggunaan Pribadi
  • Mode Terdaftar: Penggunaan Komersial

Langkah Registrasi:

  1. Buka Bantuan -> Masukkan Serial
  2. Masukkan serial key Anda
  3. Tekan "Daftarkan"
  4. Jika berhasil, HeatMon akan membuat file lisensi (reg) di folder aplikasi dan beralih ke mode "Penggunaan Komersial".
