Change Log

Version history and updates for HeatMon

  • HeatMon 1.4

    09/09/2025
    • Completely redesigned the interface to be lighter and compatible with older systems.
    • Repairing unreadable sensors on AMD Athlon Silver 3050u.

  • HeatMon 1.3

    10/12/2020
    • Fixing bugs that often appear after updating Windows 10.
    • Added a cooling system indicator feature that was previously based on FAN RPM.
    • Adding a method for parsing temperature from WMI if the sensor does not find the temperature on the CPU core.

  • HeatMon 1.2

    18/02/2019
    • Changing the temperature arrangement to be more compact and organized than a vertical list.

  • HeatMon 1.1

    12/08/2018
    • Phase 2 development trial stage

  • HeatMon 1.0

    30/05/2018

    Initial Release
