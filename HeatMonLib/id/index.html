Dokumentasi HeatMonLib

Pendahuluan

HeatMonLib adalah perpustakaan .NET profesional berbayar untuk memantau tegangan dan suhu CPU pada sistem Windows. Library ini menyediakan pengambilan data per-core, perkiraan tegangan, suhu, dan jumlah core untuk integrasi ke dalam aplikasi. Library ini cocok untuk penggunaan komersial dengan aktivasi lisensi.

Persyaratan Sistem

  • Sistem Operasi: Windows Vista / 7 / 8 / 10 / 11
  • .NET Framework 2.0 atau lebih baru
  • CPU dengan dukungan pemantauan per-core

Memulai

  1. Tambahkan referensi ke HeatMonLib.dll di proyek C# Anda
  2. Impor namespace: using HeatMonLib;
  3. Inisialisasi library dengan lisensi yang telah dibeli:
    4. string userName = "JohnDoe";
string serialKey = "A1B2C3D4E5F6G7H8I9J0"; // Diberikan setelah pembelian
HeatMonLib lib = new HeatMonLib(userName, serialKey);
  4. Jika lisensi tidak valid, akan muncul UnauthorizedAccessException
  5. Gunakan API yang tersedia untuk mengambil informasi CPU

Fitur Utama

  • Pemantauan clock CPU per-core (MHz)
  • Pemantauan tegangan CPU perkiraan (V / mV)
  • Pemantauan suhu CPU per-core (°C)
  • Mendapatkan jumlah total core CPU
  • Ringan dan mudah diintegrasikan dalam aplikasi .NET
  • Mendukung lisensi komersial

Referensi API

  • UpdateAll() - Memperbarui semua data pemantauan CPU
  • GetCpuName() - Mengembalikan nama model CPU
  • GetCpuClocks() - Mengembalikan kecepatan clock per-core dalam MHz
  • GetCpuVoltages() - Mengembalikan tegangan perkiraan per-core dalam Volt
  • GetCpuVoltages_mV() - Mengembalikan tegangan perkiraan per-core dalam millivolt
  • GetCpuTemps() - Mengembalikan suhu per-core dalam Celsius
  • GetCpuCoreCount() - Mengembalikan jumlah total core CPU
  • Close() - Melepaskan resource library

Contoh Penggunaan

using System;
using System.Collections.Generic;
using HeatMonLib;

class Program
{
    static void Main()
    {
        string userName = "JohnDoe";
        string serialKey = "A1B2C3D4E5F6G7H8I9J0";

        try
        {
            HeatMonLib lib = new HeatMonLib(userName, serialKey);

            Console.WriteLine("CPU: " + lib.GetCpuName());
            Console.WriteLine("Core: " + lib.GetCpuCoreCount());

            List<float> clocks = lib.GetCpuClocks();
            List<double> volts = lib.GetCpuVoltages();
            List<float> temps = lib.GetCpuTemps();

            for (int i = 0; i < clocks.Count; i++)
            {
                Console.WriteLine($"Core {i}: {clocks[i]} MHz | {volts[i]:F2} V | {temps[i]} °C");
            }

            lib.Close();
        }
        catch (UnauthorizedAccessException ex)
        {
            Console.WriteLine("Kesalahan lisensi: " + ex.Message);
        }
    }
}

Lisensi dan Registrasi

HeatMonLib adalah library berbayar untuk penggunaan komersial. Secara default, library ini tidak dapat digunakan tanpa lisensi yang valid.

  • Penggunaan Pribadi: Hanya untuk percobaan atau pengembangan
  • Penggunaan Komersial: Memerlukan registrasi dengan serial key yang valid

Langkah-langkah Registrasi:

  1. Beli lisensi komersial melalui: PayPal
  2. Terima serial key Anda
  3. Inisialisasi library dengan nama dan serial key Anda
  4. Jika validasi berhasil, library terbuka sepenuhnya untuk penggunaan komersial

Format Contoh Serial Key: XXX-XXX-XXX-XXX

Pemecahan Masalah

  • Tidak ada data CPU? - Pastikan CPU mendukung pemantauan per-core
  • Library melempar exception? - Periksa kembali serial key yang dimasukkan
  • Program tiba-tiba berhenti? - Jalankan aplikasi dengan hak administrator
