HeatMonLib Documentation

Introduction

HeatMonLib is a professional, paid .NET library for monitoring CPU voltage and temperature on Windows systems. It provides developers with per-core data retrieval, estimated voltages, temperatures, and core count information for integration into applications. This library is suitable for commercial usage with license activation.

System Requirements

  • Operating System: Windows Vista / 7 / 8 / 10 / 11
  • .NET Framework 2.0 or later
  • CPU with per-core monitoring support

Getting Started

  1. Add a reference to HeatMonLib.dll in your C# project
  2. Import the namespace: using HeatMonLib;
  3. Initialize the library with your purchased license:
    4. string userName = "JohnDoe";
string serialKey = "A1B2C3D4E5F6G7H8I9J0"; // Provided after purchase
HeatMonLib lib = new HeatMonLib(userName, serialKey);
  4. If the license is invalid, an UnauthorizedAccessException is thrown
  5. Use the available APIs to retrieve CPU information

Key Features

  • Per-core CPU clock monitoring (MHz)
  • Estimated CPU voltage monitoring (V / mV)
  • Per-core temperature monitoring (°C)
  • Retrieve total number of CPU cores
  • Lightweight and easy integration in .NET applications
  • Commercial license support

API Reference

  • UpdateAll() - Refreshes all CPU monitoring data
  • GetCpuName() - Returns CPU model name
  • GetCpuClocks() - Returns per-core clock speeds in MHz
  • GetCpuVoltages() - Returns estimated per-core voltages in Volts
  • GetCpuVoltages_mV() - Returns estimated per-core voltages in millivolts
  • GetCpuTemps() - Returns per-core temperatures in Celsius
  • GetCpuCoreCount() - Returns total number of CPU cores
  • Close() - Releases library resources

Example Usage

using System;
using System.Collections.Generic;
using HeatMonLib;

class Program
{
    static void Main()
    {
        string userName = "JohnDoe";
        string serialKey = "A1B2C3D4E5F6G7H8I9J0";

        try
        {
            HeatMonLib lib = new HeatMonLib(userName, serialKey);

            Console.WriteLine("CPU: " + lib.GetCpuName());
            Console.WriteLine("Cores: " + lib.GetCpuCoreCount());

            List clocks = lib.GetCpuClocks();
            List volts = lib.GetCpuVoltages();
            List temps = lib.GetCpuTemps();

            for (int i = 0; i < clocks.Count; i++)
            {
                Console.WriteLine($"Core {i}: {clocks[i]} MHz | {volts[i]:F2} V | {temps[i]} °C");
            }

            lib.Close();
        }
        catch (UnauthorizedAccessException ex)
        {
            Console.WriteLine("License error: " + ex.Message);
        }
    }
}
