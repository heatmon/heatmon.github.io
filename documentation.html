Introduction

HeatMon is a lightweight monitoring tool designed to track CPU temperature and voltage in real time. It provides live graph visualization, reporting options, and customizable themes to enhance usability and professional appearance.

System Requirements

  • Operating System: Windows Vista / 7 / 8 / 10 / 11
  • CPU: x86/x64 with temperature sensor support

Getting Started

  1. Run HeatMon.exe
  2. The main window displays:
    • CPU name and number of cores
    • Per-core temperature (°C or °F)
    • CPU voltage (Volt or mV)
    • Real-time temperature graph

Key Features

  • Real-time CPU temperature monitoring per core
  • CPU voltage monitoring (Volt / mV)
  • Interactive temperature graph with 12 color themes
  • Export reports (TXT) and graphs (PNG)
  • "Stay on Top" mode for continuous visibility
  • Personal or Commercial use licensing
  • Keyboard shortcuts for faster access

Menus and Controls

File:

  • Save Report TXT -> Save temperature/voltage report
  • Save Graph PNG -> Export graph as PNG
  • Exit -> Close HeatMon

View:

  • Show Unlisted Core -> Display additional core slots
  • Stay on Top -> Keep HeatMon on top of windows
  • Graph Themes -> Switch graph color themes
  • Show Graph -> Display graph

Help:

  • Content -> Open this help file
  • About -> Application info & credits
  • Enter Serial Key -> Open registration form

Options:

  • Voltage: Volt or mV
  • Temperature: °C or °F

Registration:

  • Enter key -> Paste/enter serial key from clipboard
  • Register -> Activate commercial license
  • Contact -> Email support for licensing issues

Keyboard Shortcuts

  • F1 -> Open Help Documentation
  • Ctrl + S -> Save Report (TXT)
  • Ctrl + Alt + S -> Save Graph (PNG)
  • Ctrl + G -> Show / Hide Graph

Data Interpretation

  • CPU Name -> Processor model
  • Core Count -> Number of cores detected
  • Voltage -> CPU voltage (Volt or mV)
  • Temps -> Per-core temperature readings

Usage Tips

  1. Enable "Stay on Top" when gaming or benchmarking
  2. Save reports periodically for performance comparison
  3. Take caution if CPU temperature exceeds 90 °C
  4. Use shortcuts for quicker workflow
  5. Register to unlock Commercial Use mode

Troubleshooting

  1. No temperature data ?
    • Ensure CPU supports temperature sensors.
  2. Fan RPM not detected ?
    • Not all systems expose fan speed sensors.
  3. Graph is empty ?
    • Wait a few seconds for data to populate.
  4. Application closes unexpectedly ?
    • Run HeatMon as Administrator.
  5. Registration failed ?
    • Verify the serial key and retry.

License and Registration

  • Default Mode: Personal Use Only
  • Registered Mode: Commercial Use

Registration Steps:

  1. Open Help -> Enter Serial Key
  2. Enter your serial key
  3. Press "Register"
  4. Upon success, HeatMon will create a license file (reg) in the application folder and switch to "Commercial Use" mode.
